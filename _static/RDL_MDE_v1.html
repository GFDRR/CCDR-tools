GitHub Authentication:

To publish datasets to the RDLS catalogue, you'll need a GitHub Personal Access Token. The editor will automatically:

  1. Create a fork of the catalogue repository in your account (if needed)
  2. Commit your dataset to your fork
  3. Open a pull request for review by repository admins
🔑 Create a new token with public_repo scope only. Your token is stored only in your browser session.
